Plugin para consumir dados de países via API (WP REST CPT) e journals científicos via API BVS Saúde, normalizar e exibir via shortcodes.

== Descrição ==

O Country Pages é um plugin WordPress que permite:

1. **Consumir dados de países** via API REST personalizada
2. **Integração com API BVS Saúde** para exibição de periódicos científicos
3. **Shortcodes flexíveis** para diferentes tipos de exibição
4. **Templates customizáveis** para total controle do layout
5. **Sistema de cache** integrado para melhor performance
