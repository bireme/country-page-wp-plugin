Plugin para consumir dados de países via API (WP REST CPT).

== Descrição ==

O Country Pages é um plugin WordPress que permite:

1. **Consumir dados de países** via API REST personalizada
2. **Shortcodes flexíveis** para diferentes tipos de exibição
3. **Templates customizáveis** para total controle do layout
4. **Sistema de cache** integrado para melhor performance
